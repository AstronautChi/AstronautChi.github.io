git常用操作

发表于 2023-05-09 分类于 C/C++编程基础
git简介 和 常用操作，非常简练。

1.什么是git?

GIT是目前世界上最先进的一款免费、开源的分布式版本控制系统。
一个工作室或者一个公司团队，多人协作时，会用到git来管理代码。

2.git的分区

git分为：工作区、暂存区、版本库、远程仓库

Workspace：工作区，本地存放项目代码的地方；
Index / Stage：暂存区，临时存放改动后的代码，其实是一个文件；
Repository：仓库区(版本库)，存放稳定版本的地方，有你提交到所有版本的数据。其中HEAD指向最新放入仓库的版本；
Remote：远程仓库，托管代码的服务器，可以简单的认为是你项目组中的一台电脑用于远程数据交换(github,gitee等)

3.git工作流程

git的工作流程一般是这样的：
1.【pull】拉远端仓库代码到本地；
2.在工作区中添加、修改代码；
3.【add】将需要提交的文件放入暂存区；
4.【commit】将暂存区的文件按要求提交到本地仓库【local repository】；
5.【push】将本地仓库push到服务器远端仓库。

4.git常用操作

4.1 配置用户名和邮箱

#用户名
git config –global user.name “xxxx”
#邮箱地址
git config –global user.email xxxx@xx.com

4.2 克隆远程仓库

#克隆一个项目和它的整个代码历史(版本信息)
$ git clone https://xxx.git

4.3 状态和提交

#查看指定文件状态
git status “文件名”
#查看所有文件状态
git status
#添加所有文件到暂存区
git add
git add -A / git add –all
#提交暂存区中的内容到本地仓库 -m 提交信息
git commit -m “内容”
#push到远端仓库
git push origin HEAD:refs/for/master
