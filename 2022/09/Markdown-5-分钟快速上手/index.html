    Markdown 5 分钟快速上手

    1. Markdown是什么？

    Markdown 是一种轻量级标记语言，被用来以纯文本格式编写文档。特点是易读，易写，很适合用来写博客，记笔记等。
    文档后缀为 **.md**, **.markdown**。
    Markdown 语言在 2004 由约翰·格鲁伯（英语：John Gruber）创建。

    2. Markdown 标题格式

    使用 # 号可表示 1-6 级标题，一级标题对应一个 # 号，二级标题对应两个 # 号，以此类推。

    3. Markdown 列表

    Markdown 支持无序列表和有序列表。
    无序列表使用星号 *、加号 + 或是减号 - 作为列表标记。
    标记后加上一个空格。
    tab 键或者 2 个空格，表示增加一个缩减层级。

    显示结果如下：

    • 第一段
    • 第一节
    • 第一小节
    • 第二段
    • 第一节
    • 第三节

    因为Markdown 无法实现”多级有序列表”，所以此处使用”无序列表”上直接手写序号, 来实现类似的效果:
